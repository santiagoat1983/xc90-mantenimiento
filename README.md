# 🔧 XC90 D5 · Ficha de Mantenimiento

Aplicación web para seguimiento del plan de mantenimiento del **Volvo XC90 D5 AWD 2019**.

**👉 [Abrir app](https://TU-USUARIO.github.io/xc90-mantenimiento/)**

## Funcionalidades

- ✅ Check de tareas por intervalo de 10 000 km
- 🔧 Tareas organizadas por sistema (Motor, Frenos, Transmisión…)
- 📝 Notas por tarea (taller, factura, observaciones)
- 💬 Comentarios generales de cada revisión
- 📋 Historial de revisiones anteriores
- 🖨️ Imprimir / guardar como PDF
- 💾 Datos guardados localmente en el navegador
